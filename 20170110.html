JS对象

JavaScript 中的所有事物都是对象：字符串、数字、数组、日期，等等。

在 JavaScript 中，对象是拥有属性和方法的数据。

JavaScript 函数语法

function functionname()

{

这里是要执行的代码

}

当调用该函数时，会执行函数内的代码。

在使用 return 语句时，函数会停止执行，并返回指定的值。

JavaScript Date（日期）对象

返回当日的日期和时间:Date()

从 1970 年 1 月 1 日至今的毫秒数:getTime()

显示星期:getDay()
